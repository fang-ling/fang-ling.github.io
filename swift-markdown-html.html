A Guide to Rendering Markdown into HTML in Swift

Undergraduate  •  Guitar Player  •  Github

Why would I need to do this?

Markdown is a text format which uses funky characters like # , ** or > to create well formatted documents. Many platforms use it on a daily basis, including my website.

Now, what if you need a markdown parser for your Swift application? This is what exactly happened to me since the whole website is rendered in Swift. Well, we could just use one of the well-tested ones (which can be found on GitHub), but instead... you can also create your own version. I'm only joking. Building a markdown parser from the ground up requires much more time than writing the entire website generator. What we really going to do is to wrap the GitHub-flavored Markdown's cmark-gfm implementation.

cmark-gfm is an extended version of the C reference implementation of CommonMark, a rationalized version of Markdown syntax with a spec. The interoperability between the Swift language and C makes us create wrappers easily.

Getting Started

Instead of wrapping cmark-gfm from the system package manager, let's use the source code directly.

# Download cmark-gfm using aria2, you may wish to use wget or curl, etc.
aria2c https://github.com/github/cmark-gfm/archive/refs/tags/0.29.0.gfm.9.tar.gz
mkdir markdown-demo
cd markdown-demo
swift package init --type executable

Wrap Functions

Now we should be able to use the C API directly in Swift, here is an example Swift wrapper function to convert Markdown String to HTML String:

import CMark
/* Convert 'text' (assumed to be a UTF-8 encoded swift string) from CommonMark
* Markdown to HTML, producing a null-terminated, UTF-8-encoded c string as an
* intermediate result, transforming it to a swift string 'result_str' and free
* the intermediate result.
*
* This func is roughly follow the routines in cmark_markdown_to_html() except
* that some extensions was enabled.
*/
func cmark_markdown_to_html_with_ext(_ text : String,
_ options : CInt) -> String {
let parser = cmark_parser_new(options)
/* Extensions register */
cmark_gfm_core_extensions_ensure_registered()
/* Add extensions here */
if let ext = cmark_find_syntax_extension("table") {
cmark_parser_attach_syntax_extension(parser, ext)
}
/* Parse */
cmark_parser_feed(parser, text, text.utf8.count)
let doc = cmark_parser_finish(parser)
/* Render */
let result = cmark_render_html(doc, options, nil)
/* Free & return */
cmark_node_free(doc)
if result == nil {
fatalError("Error: cmark_render_html()!")
}
let result_str = String(cString: result!)
free(result)
return result_str
}

Here's how it works:

@main
public struct markdown_demo {
static var text = "**Hello, World!**"
public static func main() {
print(cmark_markdown_to_html_with_ext(text, CMARK_OPT_UNSAFE))
}
}

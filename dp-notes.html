1: # of distinct paths

Given a chessboard of size $m\times n$ ($m$ rows, $n$ columns), at the beginning, you are at the most southwest corner $(1,1)$. You can either go up or right one unit each time, how many distinct paths to go to the most northeast corner $(m,n)$?

SRTBOT:

  • Subproblem: $p(i,j)=$ # of distinct paths to go to point $(i,j)$
  • Relation: $p(i,j) = p(i - 1, j) + p(i, j - 1)$
  • Topological order: $i=1\to m$, $j=1\to n$
  • Base case: $p(1,h) = p(k,1) = 1\quad(k\in [1,m], h\in[1,n])$
  • Original problem: $p(m,n)$
  • Time: $O(mn)\times O(1)=O(mn)$

Related problem: Luogu P1002

2: Longest Palindromic Subsequence

Given a string s, find the longest palindromic subsequence's length in s.

A subsequence is a sequence that can be derived from another sequence by deleting some or no elements without changing the order of the remaining elements.

SRTBOT:

  • Subproblem: $l(i,j)=$ the longest palindromic subsequence of substring $s[i ... j]$ (inclusive)
  • Relation: $$ l(i,j) = \left\{\begin{aligned} \left\{\begin{aligned}2,\quad&i==j\\2+l(i+1,j-1),\quad& \rm{otherwise}\end{aligned}\right.,\quad& s[i]==[j]\\\max\{l(i+1,j),l(i,j-1)\},\quad& \rm{otherwise}\end{aligned}\right. $$
  • Topological order: $i=(|s|-1)\to 0$, $j=(i+1)\to (|s|-1)$
  • Base case: $l(i,i) = 1\quad(i\in [0,|s|))$
  • Original problem: $l(0,|s|-1)$
  • Time: $O(|s|^2)\times O(1)=O(|s|^2)$

Related problem: LeetCode 516

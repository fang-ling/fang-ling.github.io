Here is an example to encrypt messages via openssl with the public key (PKCS#8).

echo "TOP SECRET" | openssl rsautl -encrypt -inkey PKCS8.pub -pubin | base64

Here is an example to convert public keys among various key formats.

# PKCS#1 ~---> RFC 4716/SSH2
ssh-keygen -i -f pubkeyfile -m PEM
# RFC 4716/SSH2 ~---> PKCS#8
ssh-keygen -e -f pubkeyfile -m PKCS8

You may wish to use the following command to retrieve the SHA256/MD5 fingerprint:

# SHA256
ssh-keygen -lf pubkeyfile
# MD5
ssh-keygen -E md5 -lf pubkeyfile
